# Cognifyz Restaurant Analysis

## Project Overview
This project analyzes restaurant data using Python, Pandas, NumPy, and Matplotlib.

## Tasks Completed
- Data Exploration and Cleaning
- Restaurant Rating Analysis
- Cuisine Analysis
- Geospatial Analysis
- Table Booking and Online Delivery Analysis
- Price Range Analysis
- Feature Engineering

## Tools Used
- Python
- Pandas
- NumPy
- Matplotlib
- Jupyter Notebook

## Key Findings
- Price Range 1 is the most common category.
- Higher price ranges generally receive higher ratings.
- Restaurants with table booking have better average ratings.
- Online delivery is more common among mid-range restaurants.
- Dark Green restaurants have the highest ratings.
